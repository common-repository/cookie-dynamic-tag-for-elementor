== Description ==
Enables you to use browser cookies as dynamic tags in Elementor Pro.

Note: The plugin requires Elementor Pro since it uses Dynamic Tags which is only available on the Pro version.

== Installation ==
1. To install the plugin, use the plugin manager in the WordPress administration. Go to *Plugins Â» Add New* and search for *Cookie Dynamic Tag for Elementor*.
2. Install and activate the plugin.
3. You can now find the Cookie dynamic tag under Site group inside Elementor.
